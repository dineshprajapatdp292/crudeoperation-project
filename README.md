# Student Management System using Python and SQLite

A simple **CRUD (Create, Read, Update, Delete)** based **Student Management System** built using **Python** and **SQLite3**.  
This project allows users to manage student records directly from the terminal by performing operations like **adding**, **viewing**, **searching**, **updating**, and **deleting** student information.

---

## 📌 Project Overview

This is a **console-based Python project** that stores student information in a SQLite database.  
It is designed to help understand how CRUD operations work with Python and databases in a real project.

The system stores the following student details:

- **ID**
- **Name**
- **Age**
- **Course**
- **Marks**

---

## 🚀 Features

- Add a new student record
- View all student records
- Search student by name
- Update student details
- Delete student record
- Automatic database and table creation
- Menu-driven console application
- Uses SQLite database for storage

---

## 🛠️ Tech Stack

- **Language:** Python
- **Database:** SQLite3
- **Concepts Used:** CRUD Operations, Functions, SQL Queries, Database Connectivity
